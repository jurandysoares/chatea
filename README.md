# chatea

A small Gitea CLI

## Steps

1. Copy `.env-sample` as `.env`. Execute: `cp .env-sample .env`
2. Edit the `.env` file. To create an authentication token, visit the path `/user/settings/applications` of your instance. For example: <https://gitea.com/user/settings/applications>
3. Install the `uv` command. Visit: <>
4. Restart your shell and run: `uv run chatea.py --help`
5. Explore the subcommands show in the previous step. For example, run: `uv run chatea.py {usernane}`
